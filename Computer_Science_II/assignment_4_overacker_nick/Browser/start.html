Welcome to the NickFaux web browser!

Q: What is NickFaux?

A: It's this Browser. Duh.

Q: How do I get started?

A: To browse the web, enter the URL of the website you want to visit in the address bar at the top of the window. Be sure to include http:// at the beginning of the address!

Q: That sounds like a hassle...

A: That's not a question.

Q: Did you do any of the extra credit challenges?

A: Well... kind of. I didn't handle any heavy-duty tags like anchor or img, but I did have this browser properly handle the BR tag and the P tag, and it treats a few other tags in a special way even though it does not parse them like a regular web browser.

For example, this page is written entirely in HTML in the file "start.html". Check it out!
